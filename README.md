# Digital Profile

Trang web này là một mẫu "Digital Profile" cho Nguyễn Thị Phương Thảo, Nhân viên marketing.

## Các file chính

- `index.html` - nội dung trang web
- `style.css` - giao diện đẹp và hiện đại
- `script.js` - hiệu ứng pháo hoa khi bấm nút

## Cách chạy trang web trên trình duyệt

1. Mở thư mục `d:\AI Agent\baitapdautien` trong File Explorer.
2. Nhấp đúp vào file `index.html`.
3. Hoặc mở trình duyệt, rồi kéo file `index.html` vào tab mới.

## Cách sử dụng

- Bấm nút `Tặng quà` để kích hoạt pháo hoa nổ tung trên màn hình.
- Nếu bạn muốn thay ảnh đại diện, hãy đặt file ảnh vào thư mục `d:\AI Agent\baitapdautien` với tên `avatar.jpg`.
- Sau đó mở lại `index.html` trong trình duyệt để xem ảnh mới.
